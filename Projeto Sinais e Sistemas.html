Exercise 3 É perceptível que, antes dos três gráficos acima, aparece uma mensagem de aviso, chamada de ComplexWarning. Explique o porque dessa mensagem e proponha uma solução para que ela não apareça.

De acordo com a documentação, essa mensagem de erro ocorre quando se tentar converter o numero complexo para real, por se ignorar sua parte imaginária. Ou seja se tentamos converter o 's' para valor real perdemos a parte imaginária, e o sinal mudaria. Podemos utilizar o seguinte código para ignorar as mensagens de warnings.

import warnings
warnings.filterwarnings('ignore')
